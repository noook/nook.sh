# Madami Platform

Madami is a comprehensive platform designed to support French non-profits in their operations and outreach.

## Features

- User management and authentication
- Project tracking and reporting
- Communication tools
- Analytics dashboard

## Technologies

- Vue.js 3
- Nuxt 3
- TypeScript
- Tailwind CSS
